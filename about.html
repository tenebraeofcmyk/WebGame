A Text Game

About:


This text based adventure game was designed to evaluate students' abilities in a manner that would hold student attention.

The intent was to serve as a learning tool that was engaging and fun.

Key features include:

  • Various Puzzles to Challenge the Target Grade Level
  • Interesting Story Line to Keep Players Engaged
  • Seamless Integration Between Interactive Rooms
